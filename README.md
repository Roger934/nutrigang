# NutriGang

Sistema web de gestión nutricional desarrollado con React, Node.js, Express y MySQL.

## Módulos principales

- Autenticación y autorización con JWT.
- Gestión de pacientes.
- Gestión de citas.
- Gestión de mediciones.
- Cálculos antropométricos.
- Asignación y consulta de dietas.

## Tecnologías

### Frontend
- React
- Vite
- React Router DOM
- Axios
- PropTypes
- Tailwind CSS

### Backend
- Node.js
- Express
- MySQL
- JWT
- bcrypt
- dotenv
- cors
